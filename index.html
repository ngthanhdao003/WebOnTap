Phần 2 CSS

  1. CSS (Cascading Style Sheets) là gì: là ngôn ngữ để tìm và định dạng các phần tử được tạo ra bởi HTML.
  2. Có 3 cách để đưa CSS vào HTML: Inline CSS, Internal CSS, External CSS. Trong đó External CSS là được sữ dụng nhiều nhất.
    • Inline CSS: Được dùng cho chính xác thẻ HTML bằng thuộc tính Style.
    • Internal CSS: Được sữ dụng trong chính trang HTML cụ thẻ là phần head và trong thẻ style.
    • External CSS: Đặt CSS vào file khác và inlcude vào HTML trong phần Head và trong thẻ link.

  3. CSS Selector: cho phép nhắm mục tiêu tới các phần tử HTML để áp dụng CSS cho chúng.

    Có 8 loại CSS selector: Basic CSS selector, descenant CSS selector, multiple CSS selector, Combination CSS selector, cibling CSS selector, pseudo CSS selector, attribute css selector.

    1. Basic CSS selector : Sữ dụng chọn element id / class / *.
    2. Descenant CSS selector: chọn hậu duệ bất kỳ của bất kỳ phần tử nào.
      • A . B : chọn bất kỳ phần tử nào là hậu duệ của A.
      • A > B : chọn phần tử trực tiếp của B.
      • A * : chọn tất cả các hậu duệ của A
    3. Multiple CSS selector: .A , .B , .C.... Để chọn nhiều phần tử class / id.
    4. Combination CSS selector: cho phép chứa cả phần tử AB. p.active
    5. Cibling CSS selector: nhắm chọn các phần tử anh chị em. A + B : liền kề. A ~ B : không giới hạn phần tử đầu.
    6. Pseudo CSS selector:
      • A:first-child chọn tới phần tử con đầu tiên (A phải có cha).
      • A:last-child chọn phần tử con cuối cùng.
      • A:only-child chọn tất cả A là con duy nhất của cha mẹ nó.
      • A:nth-child(n) chọn từng mục tiêu là con thứ n của cha mẹ nó.
      • A:nth-last-child(n) chọn phần tử là con thứ n của cha mẹ nó từ dưới lên.
      • A:not(B) chọn các phần từ A mà không phải là B.
    7. Attribute css selector: sữ dụng để chọn thuộc tính HTML.
      • A[B] bộ chọn các phần tử A có thuộc tính B. div[for]
      • A[B="C"] chọn phần tử A có thuộc tính B và giá trị C. div[for="Chocolate"]
      • A[B^="C"] chọn phần tử A có thuộc tính B và giá trị bắt đầu là C. div[for^="Cho"]
      • A[B$=""] chọn phần tử A có thuộc tính là B giá trị kết thúc là C. div[for$="late"]
      • A[B*="C"] chọn phần tử A có thuộc tính là B giá trị có chứa giá trị C. div[for*="cola"]

  6. Các dạng BOXS: có 3 loại block,inline,inline-block được quy định bởi thuộc tính Display.

    • Block: Có thẻ đặt thuộc tính Width, Height và sẽ bắt đầu một dòng mới. Mặc định chiếm toàn bộ chiều rộng.
    • Inline: Không thể đặt thuộc tính Width, Height. Chỉ chiếm đủ chiều rộng và không chiếm một dồng mới.
    • Inline-block: Có tính chất giống Inline tuy nhiên có thể đặt Width, Height như Block.

  7. Flex Box: gíup bạn có thể điều chỉnh các thành phần có trong container.

    • Thuộc tính cho flex-container:
      • flex-gab: px => khoảng cách giữa các items.
      • flex-direction: row | colum | row reverse | => colum reverse định phương của trục chính.
      • flex-wrap: nowrap | wrap | wrap reverse => cho các items xuống dòng nếu không có khoảng trống cho toàn bộ items.
      • justify-content: Căn chỉnh các items theo trục chính.
      • align-items: Căn chỉnh items theo trục chéo.
      • align-content: Căn chỉnh theo trực chéo Áp dụng với container nhiều hàng.
    • Thuộc tính cheo flex items:
      • Align-self: stretch | center | strat | end => ghi đè lên thuộc tính align-items
      • flex-grow: 1 => chỉ định items sẽ gấp bao nhiêu so với items khác.
      • flex-shrink: ngược lại so với flex-grow.
      • flex-basic: auto => chỉ độ dài ban đầu các items.
      • order: sắp xếp các items mà không cần thay đổi soure code.

  8. CSS grid: là là một trong những cách sử dụng CSS để xây dựng layout cho website. Khác với CSS Flex-box, nó là công cụ mạnh mẽ thường dùng cho việc phân chia bố cục cho toàn bộ trang web.
    • grid-template-columns: value1, value2,…;
      • Các giá trị value 1 ,2 thể hiện giá trị chiều rộng.Mỗi giá trị sẽ ứng với chiều rộng của các cột.
      • Có thẻ tạo ra bao nhiêu cột tuỳ ý.
      • grid-template-columns: 1fr 50px 1fr; Cột thứ hai được đặt width giá trị 50px. Cột đầu tiên và cột thứ 3 được đặt giá trị tương đối 1fr, hai cột này sẽ chiếm một nửa chiều rộng còn lại của grid-container.
      • Kích thước của một column hay một row nếu đặt giá trị auto thì nó sẽ chiếm khoảng không gian vừa đủ để hiển thị nội dung trong nó.
    • grid-template-rows: value1, value2,…; tương tụ như trên nhưng theo chiều ngang.
    • Gab , colum-gap , row-gap : Khoảng cách giữa các itmes.
    • Khi đã xác định được grid-item, chúng ta có thể dễ dàng đặt item vào các grid-cell trong grid-container bằng hai thuộc tính grid-column và grid-row.
      • grid-column: 1 / 2; xác định vị trí của item trong cột 1
      • grid-row: 2 / 3; xác định vị trí của item trong hàng 2
      • Cặp giá trị a / a + x hay a/ span x sẽ xác định được item đó bắt đầu từ đâu và kết thúc tại đâu. Giá trị x sẽ là số cell mà item đó chiếm trong grid-container.
    • justify-content : Thuộc tính căn chỉnh theo chiều ngang cho grid-container.
    • align-content: Thuôc tính canh chỉnh theo chiêu dọc.
    • justyfy-items,align-items: Canh chỉnh toàn bộ items bên trong mỗi cells.
    • align-self, content-self: Canh chỉnh theo từng items đè lên justify-items và align-items.

Phần 3 JavaScript

  1. JavaScript: là ngôn ngữ lập trình bậc cao hướng đối tượng. JS cho phép thêm các hiệu ứng động, xữ lý tương tác người dùng, điều khiển nội dung HTML CSS.
    • Giá trị: là đơn vị nhỏ nhất chứa thông tin có trong JavaScript.
    • Biến: được dùng để lưu trữ các giá trị và tái sữ dụng nhiều lần.
  2. 7 kiểu dữ liệu nguyên thuỷ: hay sữ dụng number, string, boolean. Ít sữ dụng: undefined, null, symbol, bigint.
  3. Cách khai báo biến:
    • Let: Dùng để khai báo các biến có gía trị có thể thay đổi được.
    • Const: Dùng để khai báo các biến có giá trị không thể thay đổi.
    • Var: Gần giống như let nhưng đã không còn sữ dụng.
  4. Câu lệnh if/else:if(){ ....} else {.....}
  5. Chuyển đổi kiểu và ép kiểu dữ liệu:
    • String -> Number : Number(ValueString);
    • Number -> String : String(Number);
    • Javascript chỉ có thể chuyển đổi thành ba kiểu: number, string, boolean.
    • Javascript sẽ chuyển đổi kiểu trong các giá trị để khớp với các giá trị khác để cuối cùng có thể thực hiện phép tính.

      '9' - '5' = 4;

  6. Gía trị boolean:

  7. Toán tử == , ===

    • == :Toán tử == được gọi là toán tử bằng lỏng lẻo vì nó thực hiện ép kiểu trước khi so sánh. Ví dụ "18" == 18 // return true.
